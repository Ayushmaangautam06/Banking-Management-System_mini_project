# 🏦 Banking Management System (Python)

## 📌 Description
This project is a console-based Banking Management System developed using Python. It simulates real-world banking operations and allows users to perform various financial activities through a menu-driven interface.

The system includes features such as account creation, secure login using PIN authentication, deposit and withdrawal of money, balance checking, loan management, investments (SIP and mutual funds), fixed deposits, and more.

The program ensures basic validation and security by limiting login attempts and maintaining a minimum account balance.

---

## 🎯 Objectives
- To simulate real-world banking operations using Python
- To implement menu-driven programs with multiple features
- To practice concepts like loops, conditionals, and exception handling
- To manage financial calculations such as interest, EMI, and investments

---

## 🚀 Features

### 🔐 Security
- PIN-based login system
- Maximum 3 login attempts
- Account blocking after failed attempts

---

### 💰 Account Operations
- Deposit money (Cash, UPI, Debit, Net Banking)
- Withdraw money (with minimum balance rule)
- Check account details and balance
- Track number of transactions

---

### 📈 Financial Calculations
- Interest calculation on balance
- SIP (Systematic Investment Plan)
- Mutual fund investment (Low, Medium, High risk)
- Fixed Deposit (FD) calculation
- Loan EMI calculation

---

### 🏦 Loan System
- Personal Loan
- Home Loan
- Educational Loan
- Loan repayment system

---

### 💳 Additional Features
- Money transfer
- ATM card details generation
- Minimum balance maintenance (₹500)

---

## ⚙️ Technologies Used
